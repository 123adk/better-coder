# IEEE 118节点VPP协同调度仿真修正版

## 概述

本项目是IEEE 118节点虚拟电厂(VPP)协同调度仿真的修正版本，解决了原版本中的多个关键问题，包括并行池启动失败、发电机组识别错误、算法收敛问题等。

## 主要修正内容

### 1. 并行池启动问题修正
- **问题**: 原代码尝试加载不存在的文件`calculate_generation_cost.m`
- **修正**: 移除不存在的文件依赖，改进并行池启动逻辑，增加错误处理

### 2. 发电机组识别修正
- **问题**: 所有VPP的机组数都显示为0
- **修正**: 正确从发电机数据第13列读取VPP_ID，构建正确的VPP子网络结构

### 3. 算法收敛性改进
- **问题**: 每次迭代后结果不变，收敛困难
- **修正**: 
  - 改进拉格朗日对偶算法参数设置
  - 增加自适应步长机制
  - 添加随机扰动避免局部最优
  - 修正功率平衡计算

### 4. 成本计算函数完善
- **问题**: 缺失VPP成本计算函数
- **修正**: 
  - 添加完整的VPP成本计算函数
  - 正确处理可中断负荷的负值成本
  - 支持成本明细分析

## 文件结构

```

├── solve_VPP_subproblem_QGA.m                  # 量子遗传算法VPP子问题求解

├── generate_results_analysis.m                 # 结果分析和可视化
├── test_IEEE118_VPP_simulation.m               # 测试脚本
└── IEEE118_VPP_README.md                       # 说明文档
```

## 主要特性

### 1. 分层优化架构
- 使用拉格朗日对偶分解将耦合约束松弛到目标函数中
- 分解为多个VPP子问题并行求解
- 保持量子遗传算法作为子函数调用的结构

### 2. 量子遗传算法(QGA)
- 量子比特编码表示解空间
- 量子门更新机制
- 量子测量和灾变操作
- 自适应参数调整

### 3. 成本模型
- 二次发电成本函数
- 可中断负荷负值成本处理
- 多VPP成本分摊和分析

### 4. 收敛性保证
- 自适应步长调整
- 随机扰动机制
- 收敛历史跟踪
- 多重收敛判据

## 技术参数

### VPP配置
- VPP数量: 5个
- 发电机总数: 12台
- 母线数量: 118个
- 调度时段: 24小时

### 算法参数
- QGA种群大小: 30
- QGA最大代数: 50
- 拉格朗日最大迭代: 100
- 收敛容差: 1e-4

## 故障排除

### 常见问题

1. **并行池启动失败**
   - 解决方案: 程序会自动切换到串行计算模式

2. **内存不足**
   - 解决方案: 减少QGA种群大小或VPP数量

3. **收敛困难**
   - 解决方案: 调整步长参数或增加最大迭代次数
